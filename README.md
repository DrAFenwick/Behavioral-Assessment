Behavioral Awareness Assessment – README

Project Overview

This project is an interactive Behavioral Awareness Assessment designed to help users understand their behavioral patterns, interpersonal tendencies, and work-style preferences.

It is not a clinical diagnostic tool. this is a tool to assess your own behaviors and is for self awareness and educational purposes only. 
The assessment evaluates four categories:
	1.	Interaction Style
	2.	Emotional & Energy Patterns
	3.	Decision-Making & Focus
	4.	Social & Work Preferences

Users rate themselves on a 1–5 scale using sliders, and the system provides:
	•	Category-wise scores
	•	Overall behavioral profile
	•	Strengths and growth suggestions
	•	A bar chart visualization
	•	Downloadable PDF summary

⸻

Features
	•	Mobile-friendly slider inputs for each question
	•	Real-time progress bar
	•	Dynamic scoring and category analysis
	•	Instant feedback with behavioral type and tips
	•	Bar chart visualization of category scores using Chart.js
	•	Downloadable PDF summary using jsPDF


⸻

Usage
	1.	Users slide each question to indicate how much it reflects their behavior (1–5)
	2.	Progress bar updates as questions are answered
	3.	Click Submit Assessment
	4.	Results page shows:
	•	Total score
	•	Category scores chart
	•	Behavioral profile and suggestions
	5.	Optional: click Download PDF to save a personal summary

⸻

Technologies Used
	•	HTML / CSS / JavaScript – structure and interactivity
	•	Chart.js – for category score bar chart
	•	jsPDF – for generating downloadable PDF summary

⸻

Notes
	•	Assessment is for personal awareness and educational purposes only
	•	Results are not a clinical diagnosis
	•	Always use responsibly and ethically
