Behavioral Awareness Assessment – README

Project Overview

This project is an interactive Behavioral Awareness Assessment designed to help users understand their behavioral patterns, interpersonal tendencies, and work-style preferences.

It is not a clinical diagnostic tool. this is a tool to assess your own behaviors and is for self awareness and educational purposes only. 

# Behavioral Awareness Assessment

## Project Overview
An interactive self-reflective assessment to explore behavioral tendencies. 
Not for clinical diagnosing. 

**Categories:** Interaction, Energy, Decision-Making, Social.

**Outputs:** 
- Category scores (bar chart)  
- Overall behavioral profile  
- Suggestions for growth  
- Downloadable PDF summary  

## Launch Instructions
### Locally
- Open `index.html` in any browser.

### Online
**GitHub Pages**
- Upload `index.html` to repo → Settings → Pages → Deploy → Live URL.

**Netlify**
- Drag & drop `index.html` → Live URL generated.

**Vercel**
- Import project → Live URL generated.

## Technologies
- HTML / CSS / JS  
- Chart.js for visualization  
- jsPDF for PDF download  

## Customization
- Edit HTML for question text  
- Edit JS `calculateResults()` for scoring logic or feedback  
- Edit CSS for colors, fonts, layout
